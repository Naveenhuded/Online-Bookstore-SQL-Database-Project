# 📚 Online Bookstore SQL Database Project

## Project Overview
This project demonstrates the design and implementation of an Online Bookstore Database using PostgreSQL. It includes relational database design and SQL queries to analyze sales, customers, inventory, and revenue.

## Database Tables
- Books
- Customers
- Orders

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- Aggregate Functions (SUM, AVG, COUNT)
- DISTINCT
- LIMIT

## Business Problems Solved
- Retrieve books by genre and publication year.
- Find highest-priced books.
- Calculate total revenue.
- Identify top-selling books.
- Analyze genre-wise sales.
- Find highest-spending customers.
- Monitor inventory levels.
